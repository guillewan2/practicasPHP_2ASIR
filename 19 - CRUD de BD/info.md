# Bases de Datos en PHP
Para comunicar PHP con un DBMS (Database Management System), necesitamos un driver para que se comunique con MySQL por su protocolo. PHP tiene un driver para MySQL, SQLite y para otras DMBS. Las de MySQL y SQLite vienen por defecto instaladas en el sistema. 

Las estructuras de clase de Programación Orientada a Objetos se puede mapear facil con tablas de bases de datos relacionales. Una forma de estructurar una aplicación web que funcione con una base de datos es diseñar una clase que corresponda a cada tabla de la base de datos. Esas clases tendrían propiedades que reflejen las columnas de la base de datos, y una instancia de la clase correspondría a una fila de la tabla.

Si queremos escribir datos a una tabla de la base de datos, podemos crear primero el objeto de la clase que contenga nuevos datos.
