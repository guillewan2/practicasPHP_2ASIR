# 23 - Juego de Aciertos
Añadiendo la siguiente funcionalidad al ejemploa anterior. Vamos a realizar un juego en el que el usuario tendrá que introducir palabras y cuando las acierte tendrá que mostrar un mensaje de acertado. 

Ejemplo: Tengo una página web donde el usuario introduce una palabra. Esta palabra se envía a un PHP donde tengo una estructura `["hola","como","pepe","juan"]`. Tiene que comprobar si la palabra que ha introducido está en nuestra estructura. En caso de estar, habrá acertado. En cualquier otro caso, no habrá acertado.

> [!NOTE] Nota
> Si el usuario introduce HOLA, Hola, HOla,... se da como acertado también. 
> Es decir, cualquier combinación de mayúsculas y minúsculas tiene que ser considerada.
